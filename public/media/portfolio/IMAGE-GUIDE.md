# Fly Girl Portfolio Image Guide

Folders map to the website's portfolio categories. To add or replace photos,
drop files into the matching folder using descriptive kebab-case filenames
(e.g. `01-executive-dinner-tablescape.jpg`), then reference them in
`src/lib/portfolio-galleries.ts`. Keeping the same filename when replacing a
photo updates the site with no code changes.

- corporate-executive/       → Corporate & Executive Experiences (concepts live in /media/concepts)
- destination-wedding/       → Destination Experiences (featured Jamaica event)
- styled-spaces/             → Styled Spaces (awaiting photos)
- welcome-gifts/             → Welcome Gifts (currently sourced from event folders)
- marios-50th-birthday/      → Private Celebrations
- azoyahs-graduation-celebration/ → Private Celebrations
- laurens-7th-birthday/      → Little Luxe™
- little-luxe/               → Little Luxe™ (Glamp & Glow)

AI-generated visuals live in /public/media/concepts and must always be
labeled Design Concept / Styling Concept / Concept Visualization — never
presented as completed client work.

V3 concept folders (all labeled Design Concepts on the site):
- /media/concepts/sports/          → Draft Party & Team Dinner (with befores)
- /media/concepts/corporate/       → Networking event concept

Replace any concept with real photography by dropping in a file with the same name.
